# Zeaxan

> Suplemento para la salud visual con luteína, zeaxantina y omega-3 DHA basado en el estudio AREDS2.

**[Visitar zeaxan.pe](https://zeaxan.pe)**

## Sobre este proyecto

Este repositorio contiene el sitio web de [Zeaxan](https://zeaxan.pe), un recurso de salud basado en evidencia científica enfocado en **Salud Visual**.

### Ingredientes clave

Luteína (10mg), zeaxantina (2mg), omega-3 DHA, zinc, cobre, vitaminas C y E

## Tecnología

- HTML5 + CSS3 (sitio estático)
- Hospedado en GitHub Pages
- Sin dependencias de JavaScript frameworks
- Optimizado para Core Web Vitals

## Evidencia científica

Todo el contenido está respaldado por:
- Estudios clínicos publicados en PubMed
- Monografías de la EMA (European Medicines Agency)
- Guías de la OMS sobre plantas medicinales
- Base de datos TRAMIL de plantas del Caribe y Latinoamérica
